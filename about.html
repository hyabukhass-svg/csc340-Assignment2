About This Website

This website was created as a Disney princess character gallery. I wanted to practice designing a colorful, magical, and visually appealing website while learning HTML and CSS.

The color scheme (soft purple background, deep rose header, and cream navigation) is meant to create a warm, magical princess aesthetic. Fonts, card design, and consistent navigation ensure the site feels cohesive.

Each character has a gallery card on the home page and a detailed profile page with story, abilities, and personality traits.
